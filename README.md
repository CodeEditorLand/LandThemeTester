# Theme Tester

Test a theme on a set of input files without installing it.

Try it out in vscode.dev: https://vscode.dev/theme/aeschli.senja-dark

![screenshot](./screenshot.png)

Also works in desktop or web: Command: `Theme Tester: Preview Theme`

> **Note**: VS Code also has a built-in feature to try out themes:<br> Run
> `Preferences: Color Themes` and select `Browse Additional Color Themes...`
